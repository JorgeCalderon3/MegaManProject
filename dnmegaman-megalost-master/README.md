# Project 1 - Definitely Not Megaman
	Small project description	
	This project is a slight adaptation of the famous franchise Megaman.
	It has 5 levels. Some prove to be a little challenging while others are more
	simple. It has a boss at the final level and a power up in the third level.

	Estimated Time spent: About 15 - 20 hrs
